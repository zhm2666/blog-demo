# 新手入门指南 - Gin-Vue-Blog 全栈项目

## 一、项目简介

Gin-Vue-Blog 是一个**前后端分离**的**全栈博客系统**，采用当前最火、最新的技术栈开发。代码轻量级、注释完善，非常适合新手学习。

### 1.1 核心特性

| 特性 | 说明 |
|------|------|
| 前后端分离 | 前端 Vue3 + 后端 Go Gin |
| 权限管理 | RBAC 权限模型，支持动态菜单 |
| 博客功能 | 文章、分类、标签、评论、留言 |
| 技术新颖 | Go 1.19 + Vue3 + TypeScript + Pinia |
| 部署简单 | Docker Compose 一键部署 |

## 五、关键文件解读

### 5.1 后端入口 (main.go)

```go
// gin-blog-server/cmd/main.go

func main() {
    // 1. 读取配置
    conf := g.ReadConfig(*configPath)

    // 2. 初始化数据库、Redis、日志
    db := ginblog.InitDatabase(conf)
    rdb := ginblog.InitRedis(conf)

    // 3. 配置 Gin 中间件
    r.Use(middleware.CORS())        // 跨域
    r.Use(middleware.WithGormDB(db)) // 数据库
    r.Use(middleware.WithRedisDB(rdb)) // Redis
    r.Use(middleware.JWTAuth())      // JWT认证
    r.Use(middleware.PermissionCheck()) // 权限验证

    // 4. 注册路由
    ginblog.RegisterHandlers(r)

    // 5. 启动服务
    r.Run(serverAddr)
}
```

### 5.4 前端路由守卫

```javascript
// gin-blog-admin/src/router/guard.js

router.beforeEach(async (to) => {
    const { token } = useAuthStore()

    // 没有 Token，重定向到登录页
    if (!token) {
        if (['/login', '/404'].includes(to.path)) {
            return true
        }
        return { name: 'Login', query: { redirect: to.path } }
    }

    // 已登录不能访问登录页
    if (to.name === 'Login') {
        return { path: '/' }
    }

    return true
})
```

## 七、新手实战任务

### 任务1：添加一个新页面

1. **后端**：在 `handle/` 新建 `handle_hello.go`
2. **路由**：注册新路由
3. **前端**：在 `views/` 新建 `hello/index.vue`
4. **菜单**：在数据库添加菜单记录

### 任务2：修改文章标题

1. 找到 `handle_article.go` 中的 Update 方法
2. 修改对应的数据库更新逻辑
3. 重启后端服务测试

### 任务3：添加一个新的 API

1. 在 `handle/` 写处理函数
2. 在路由文件中注册
3. 在 `resource` 表添加资源记录
4. 给角色分配该资源权限

### 任务4：理解权限流程

1. 登录获取 Token
2. 带着 Token 请求受保护的资源
3. 中间件验证 Token 和权限
4. 返回结果或 403 错误
